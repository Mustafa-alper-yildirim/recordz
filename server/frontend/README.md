# Frontend Hazir Yapisi

Bu klasor, mevcut statik panelin React veya Next.js surumune tasinmasi icin ayrilmistir.

## Onerilen Yapilar

- `src/pages/Login.jsx`
- `src/pages/Dashboard.jsx`
- `src/pages/Cari.jsx`
- `src/pages/Offers.jsx`
- `src/pages/Finance.jsx`
- `src/lib/api.js`
- `src/context/AuthContext.jsx`

## Hazir Dosyalar

- `src/lib/api.js`: Backend API istemcisi
- `src/context/README.md`: Auth context akisi notlari

## Not

Kok dizindeki statik panel de benzer sekilde `api-client.js` ile backend'e baglanmistir.
