# Server Hazir Yapisi

Bu klasor, uygulamayi tarayici icinden gercek sunuculu yapiya tasimak icin hazirlanmis iskeleti icerir.

## Hedef Mimari

- `backend/`: Node.js + Express + SQLite/PostgreSQL API
- `frontend/`: React veya Next.js istemci
- Gercek kullanici girisi
- Rol bazli yetkilendirme
- JWT tabanli oturum yonetimi

## Not

Bu ortamda `Node.js` kurulu olmadigi icin dosya yapisi hazirlandi ancak calistirilmadi.
